Prework Study Guide

HTML

  • The head element contains metadata, information that the web browser uses to interpret the page.
  • The body element contains what the user is intended to see.
  • There should only ever be one h1 element on a page.
  • the id attribute is used to identify elements
  • the header element is used at the top of the page, and the footer at the bottom
  • section elements are used to organize the page

CSS

  • margin is the space around the element, while padding is space within
  • class selectors always begin with '.'
  • * is a wildcard selector to apply styles to all visible elements.
  • external stylesheets can be linked in the head section of the html document

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: adds All changes to staging area
  • git commit -m "message" : commits staged changes to current branch
  • git pull origin main: pulls main branch to make sure it is up to date
  • git push origin branch-name: pushes local branch changes to github
